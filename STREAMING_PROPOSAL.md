## Какие данные собираются

### 1. Тикер (каждую секунду) — уже в stream.db `tickers`

```
Цена, изменение за 24ч, high/low, объём, оборот
```

### 2. Стакан заявок (orderbook) — stream.db `orderbook_snapshots`

| Метрика | Что показывает | Как использовать |
|---------|---------------|-----------------|
| **Спред** | bid-ask разница | Узкий = ликвидный рынок, хороший вход |
| **Глубина ±1%** | Сколько TON в пределах 1% от цены | Тонкий стакан = цена легко сдвинется |
| **Дисбаланс** | bid_depth / ask_depth | >1.5 = покупатели давят (цена ↑), <0.67 = продавцы (цена ↓) |
| **Стены** | Скопления >5000 TON | Где крупные игроки поставили ордера |

### 3. История сделок — stream.db `trades_stream`

| Метрика | Что показывает |
|---------|---------------|
| **Buy/Sell ratio** | % покупок от объёма за 5 мин |
| **Крупные сделки** | >$1000 = кит активен |

### 4. Что можно добавить из Bybit REST API (бесплатно, без ключа)

| Эндпоинт | Данные | Зачем |
|----------|--------|-------|
| `/v5/market/open-interest` | Открытый интерес | Растёт = новый капитал входит |
| `/v5/market/funding-rate` | Ставка финансирования | Положительная = перегретый лонг |
| `/v5/market/insurance` | Страховой фонд | Растёт = риск ликвидаций |
| `/v5/market/kline?interval=D` | Дневные свечи | Для долгосрочного тренда |

## Как стриминг улучшит сигналы (v6)

### Сейчас (v5, 1 мин)
```
Cron раз в минуту → RSI по минутным свечам → сигнал
Реакция: до 60 секунд задержки
```

### После (v6, 1 сек)
```
WebSocket → поток цен → RSI по секундным ценам → мгновенный сигнал
Реакция: < 1 секунды
```

### Новые фильтры из стакана

| Фильтр | Логика |
|--------|--------|
| **Дисбаланс > 2x** | Усилить сигнал на 1 уровень (WEAK→STRONG) |
| **Спред > 0.1%** | Пропустить сигнал (неликвидный момент) |
| **Стена на уровне SL** | Сдвинуть SL за стену (защита от охоты на стопы) |
| **Buy/Sell ratio > 80%** | Подтверждение BUY-сигнала реальными сделками |

### Точность SL/TP

Вместо фиксированных 0.8% от цены:
- SL ставится **за стеной покупок** (если стена на 2.1300, SL чуть ниже)
- TP упирается **в скопление асков** (если стена продаж на 2.1600, TP чуть ниже)
