# TradingView — интеграция как с сервисом

## 📌 Как работает TV (когда ты открываешь его в браузере)

1. Ты заходишь на tradingview.com → график TONUSDT
2. Добавляешь индикатор (Pine Script) — или берёшь из сообщества
3. TV сам тянет данные с бирж (Bybit в том числе)
4. Индикатор считает всё на клиенте (в браузере)
5. Если настроен alert → TV шлёт webhook на твой URL

**Ключевой момент:** TV не умеет отправлять данные наружу постоянно. Только по триггеру алерта.

### 1. Webhook алерты (✅ работает, бесплатно с аккаунтом)

Самый простой способ: создать alert на TV, который дёргает наш сервер.

**Пошагово:**

1. На TV открываешь график TONUSDT
2. Добавляешь индикатор (например, «RSI Strategy» из сообщества)
3. Нажимаешь на колокольчик (Alert) → Webhook URL
4. В поле URL вписываешь: `http://oleg-ai:8765/tv-webhook`
5. В поле Message вставляешь JSON:
```json
{
  "symbol": "{{ticker}}",
  "price": "{{close}}",
  "time": "{{time}}",
  "timeframe": "{{interval}}",
  "action": "{{strategy.order.action}}",
  "comment": "{{strategy.order.comment}}"
}
```
6. Сохраняешь

**Что произойдёт:** когда индикатор на TV даст сигнал → наш сервер получит JSON → отправит тебе в Telegram.

| Плюсы | Минусы |
|-------|--------|
| ✅ Бесплатно (1 алерт на free-аккаунте) | ❌ Только 1 алерт на free tier |
| ✅ Готовые индикаторы сообщества | ❌ В Pro ($15/мес) — 30 алертов |
| ✅ Ты видишь график и индикатор глазами | ❌ Не влияет на нашу стратегию автоматически |
| ✅ 5 минут настройки | |

### 2. TV скринер → фильтр монет (✅ бесплатно)

TV умеет сканировать ВЕСЬ рынок по твоим критериям и показывать сигналы.

**Пример:** «Покажи все монеты где RSI < 30 на 1h + объём > $1M»

```pinescript
// Скопируй это в Pine Script на TV (новый скрипт → Screener)
rsi_val = ta.rsi(close, 14)
vol_filter = volume > sma(volume, 20) * 1.5

signal = rsi_val < 35 and close > ta.sma(close, 20) and vol_filter

alertcondition(signal, title="BUY Signal", message="RSI oversold + trend up")
```

Это не даст сигналы в нашего бота напрямую, но ты увидишь их на TV в реальном времени.

**Что это даёт:** Ты мониторишь НЕ только TON, а **весь рынок** (1000+ монет) через один экран TV. Когда видишь интересный сигнал — говоришь мне, и я анализирую эту монету.

| Плюсы | Минусы |
|-------|--------|
| ✅ Бесплатно | ❌ Не автоматизировано (ты сам смотришь) |
| ✅ 1000+ монет одновременно | ❌ Нет интеграции с нашим ботом |
| ✅ Можно быстро найти точку входа | |
