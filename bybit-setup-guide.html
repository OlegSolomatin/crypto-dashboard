🚀 Фаза 4: Подключение Bybit Testnet

3 простых шага — и трейдер начнёт торговать виртуальными деньгами
ШАГ 1
Регистрация на Testnet
Открой в браузере (без VPN):
https://testnet.bybit.com

Зарегистрируйся: email + пароль.
Это тестовая сеть — все деньги виртуальные.
Тебе автоматически начислят 100,000 USDT для тестов.
ШАГ 2
Создание API-ключей
После регистрации в Testnet:

① Наведи на иконку профиля (правый верхний угол) → API
② Нажми «Create New Key» → выбери «System-generated API Key»
③ Настройки:
   • API Transaction: ОТКЛЮЧИТЬ
   • Trade (SPOT + Futures): ВКЛЮЧИТЬ
   • Read-Write: ВКЛЮЧИТЬ
   • IP-ограничения: оставь пустыми (никаких)
④ Сохрани (скопируй!) API Key и API Secret
ШАГ 3
Отправь ключи мне в чат
Просто напиши сюда:
API_KEY=твой_api_key
API_SECRET=твой_api_secret

Я пропишу их в .env, активирую trader.py через cron,
и система начнёт торговать на тестнете.

✅ Что будет после настройки

  • signal_v5.py отправляет сигнал BUY/SELL
  • trader.py исполняет ордер на Testnet (виртуальные деньги)
  • Стоп-лосс и тейк-профит ставятся автоматически
  • Уведомление в Telegram: «✅ Сделка открыта»
  • Первые 100 сделок — ТОЛЬКО тестнет
