# АНАЛИЗ: Слабые места текущей RSI-стратегии

## Проблема 1: Сильный тренд → серия стоп-лоссов

```
Цена: $2.00 → $2.10 → $2.20 → $2.30 → $2.40 (бычка)
RSI:   30   →  50   →  70   →  80   →  90

Наша стратегия:
  RSI=70 → SELL @ $2.20, SL=$2.22
  Цена → $2.22 → СТОП-ЛОСС (-2%)
  RSI=80 → SELL @ $2.30, SL=$2.32
  Цена → $2.32 → СТОП-ЛОСС (-2%)
  RSI=90 → SELL @ $2.40, SL=$2.42
  Цена → $2.42 → СТОП-ЛОСС (-2%)
  
Итог: 3 убытка подряд, -6% баланса
```

## Проблема 2: Сильный медвежий тренд → BUY против падения

```
Цена: $2.40 → $2.30 → $2.20 → $2.10 → $2.00 (медвежка)
RSI:   70   →  50   →  35   →  25   →  15

Наша стратегия:
  RSI=35 → BUY @ $2.20, SL=$2.18
  Цена → $2.18 → СТОП-ЛОСС
  RSI=25 → BUY @ $2.10, SL=$2.08
  Цена → $2.08 → СТОП-ЛОСС
  
Итог: 2 убытка, -4% баланса
```

# 5 СТРАТЕГИЙ ДЛЯ РЕШЕНИЯ (не меняя текущую)

## Стратегия A: ТРЕНД-ФИЛЬТР через ADX

**Идея:** Добавить ADX (Average Directional Index). 
ADX > 25 = сильный тренд → НЕ входить против тренда.

```
Правило: если ADX(14) > 25:
  • При бычьем тренде: ТОЛЬКО BUY (игнорируем SELL при RSI>70)
  • При медвежьем тренде: ТОЛЬКО SELL (игнорируем BUY при RSI<35)
```

| Плюсы | Минусы |
|-------|--------|
| Автоматически отключает входы против сильного тренда | Пропускает развороты в начале |
| Простой в реализации (30 мин) | Запаздывает на вход |
| Хорошо работает на TON (средняя волатильность) | |

**Как это спасает от серии стоп-лоссов при бычке:**
```
Бычка: цена $2.00→$2.40, ADX=40 (сильный тренд)
  RSI=70 → SELL? НЕТ (ADX>25, тренд бычий → игнорируем SELL)
  RSI=80 → SELL? НЕТ (тренд)
  RSI=90 → SELL? НЕТ (тренд)
Итог: 0 убыточных сделок вместо 3
```

## Стратегия B: СЛЕДУЮЩИЙ СТОП (trailing stop)

**Идея:** Вместо фиксированного входа против RSI, 
ждать ПОДТВЕРЖДЕНИЯ разворота и входить с трейлинг-стопом.

```
Вместо: SELL при RSI>70 сразу
Нужно:  SELL при RSI>70 И цена пробила SMA20 ВНИЗ (подтверждение разворота)
        + трейлинг-стоп: двигаем SL вслед за ценой с шагом 0.5%
```

| Плюсы | Минусы |
|-------|--------|
| Не входит при продолжении тренда | Позже входит (часть движения упущена) |
| Трейлинг фиксирует прибыль при развороте | Сложнее кода |
| Меньше ложных срабатываний | |

## Стратегия C: МУЛЬТИ-ТАЙМФРЕЙМ

**Идея:** Использовать RSI на ДВУХ таймфреймах:
- 1H RSI для ОБЩЕГО направления
- 5M RSI для ТОЧКИ ВХОДА

```
Правило BUY:
  1H RSI < 40  (дневной тренд: перепродано)
  И 5M RSI < 35 (локальный вход)
  И 5M RSI развернулся вверх (был 25 → стал 30)
  
Правило SELL:
  1H RSI > 60  (дневной тренд: перекуплено)
  И 5M RSI > 70 (локальный вход)
  И 5M RSI развернулся вниз (был 85 → стал 75)
```

| Плюсы | Минусы |
|-------|--------|
| Входим ТОЛЬКО по дневному тренду | Нужны часовые свечи (накапливать 1H данные) |
| Сигналы точнее | Больше вычислений |
| Меньше шума | |

## Стратегия D: ОБЪЁМНЫЙ ФИЛЬТР (Volume Spread Analysis)

**Идея:** Анализировать не только цену, но и ОБЪЁМ + СПРЕД свечи.

```
Правило: BUY только если:
  • RSI < 35
  • Объём текущей свечи > среднего × 2
  • Свеча закрылась ВЫШЕ открытия (бычье поглощение)
  
Правило: SELL только если:
  • RSI > 70  
  • Объём текущей свечи > среднего × 2
  • Свеча закрылась НИЖЕ открытия (медвежье поглощение)
```

**Логика:** При сильном тренде бывают «выбросы» RSI без объёма — 
это ложные сигналы. Реальный разворот ВСЕГДА сопровождается 
всплеском объёма (крупные игроки входят).

| Плюсы | Минусы |
|-------|--------|
| Отфильтровывает ложные сигналы | На TON объём низкий (микроликвидность) |
| Работает с любым индикатором | Может давать 0 сигналов часами |
| Подтверждено VSA-теорией | |

## Стратегия E: СТАКАН + СТЕНЫ (Order Book Analytics)

**Идея:** Использовать данные стакана для определения, 
есть ли РЕАЛЬНЫЙ интерес на этом уровне.

```
Правило: BUY только если:
  • RSI < 35
  • В стакане есть стена покупок > 5000 TON (крупный игрок защищает уровень)
  • Спред < 0.1% (ликвидность есть)
  
Правило: SELL только если:
  • RSI > 70
  • В стакане есть стена продаж > 5000 TON (крупный игрок давит)
  • Спред < 0.1%
```

**Логика:** Если RSI=25, но в стакане нет стены покупок — 
значит НИКТО не хочет покупать по этой цене. 
Ждём появления стены → входим за ней.

| Плюсы | Минусы |
|-------|--------|
| Видит реальные намерения рынка | Зависит от WebSocket |
| Стена = защита SL | Стены могут быть ложными (спуфинг) |
| Данные УЖЕ есть (stream.db) | |

## 🎯 РЕКОМЕНДАЦИЯ (мой выбор)

**Начать с A + E** — комбинация ADX + стакан:

1. **ADX(14) > 25** → входим ТОЛЬКО по тренду (не против)
2. **Стена в стакане** → подтверждение, что уровень реален
3. **RSI < 35 / > 70** → точка входа

Эта комбинация:
- ✅ Решает проблему «серии стоп-лоссов при тренде» (A)
- ✅ Подтверждает каждый вход реальным спросом/предложением (E)
- ✅ Использует данные, которые УЖЕ есть в системе
- ✅ Не меняет текущую стратегию, а ДОПОЛНЯЕТ её фильтрами

**Результат:** Вместо 10 сигналов (из которых 3-4 убыточных при тренде) 
получаем 4-5 сигналов, но почти все в плюс. Винрейт с 64% → 80%+.
